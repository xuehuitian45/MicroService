# 微服务节点依赖关系可视化

这个工具用于可视化展示微服务中节点之间的依赖关系。

## 功能特点

- 📦 **服务分组显示**：每个服务显示为一个独立的框
- 🔗 **依赖关系连线**：节点之间的依赖关系用线条连接
- 🎨 **颜色区分**：不同服务使用不同颜色，便于区分
- 🔍 **交互功能**：
  - 鼠标悬停查看节点详细信息
  - 缩放和平移视图
  - 切换节点标签显示/隐藏
  - 重置视图

## 使用方法

### 方法一：使用Python HTTP服务器（推荐）

1. 在项目根目录运行服务器：
```bash
cd /Users/xht/Downloads/MicroService
python visualize/server.py
```

2. 在浏览器中访问：
```
http://localhost:8000/visualize/index.html
```

### 方法二：直接打开HTML文件

如果浏览器允许本地文件访问，可以直接双击 `visualize/index.html` 文件。

**注意**：某些浏览器可能因为安全策略无法直接加载本地JSON文件。如果遇到问题，请使用方法一。

## 数据文件

可视化工具从以下文件读取数据：

- `data/data.json` - 节点信息和依赖关系
- `result/result.json` - 服务分组信息

## 界面说明

- **左侧控制面板**：包含重置视图和切换标签按钮
- **右侧图例**：显示所有服务及其颜色
- **主视图区域**：
  - 每个服务显示为一个带颜色的框
  - 框内包含该服务的所有节点（圆形）
  - 节点之间的依赖关系用线条连接
  - 鼠标悬停在节点上可查看详细信息

## 操作说明

- **缩放**：使用鼠标滚轮
- **平移**：按住鼠标左键拖动
- **查看详情**：将鼠标悬停在节点上
- **重置视图**：点击"重置视图"按钮
- **切换标签**：点击"切换标签"按钮显示/隐藏节点名称
